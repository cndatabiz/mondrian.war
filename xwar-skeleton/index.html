Welcome to X-Mondrian!

X-Mondrian is a distribution of the Mondrian (R)OLAP engine. It combines

  • Binaries and documentation built from the open source Mondrian project
  • Configuration files to control mondrian operation and logging
  • Configuration files to deploy mondrian to a servlet container (like Tomcat, Jetty, etc.)
  • A web-client library to access mondrian using XML/A (along with resourcs like API documentation, code samples)
  • Sample data and a few actual mondrian web-client applications
X-Mondrian packages all these items into a single .WAR (web archive) file which is then easily deployable to a servlet container, like Apache Tomcat, Eclipse Jetty or Tiny Webserver (to name just a few).

The purpose of X-Mondrian is to quickly get up and running with a fully functional Mondrian instance that is accessible through a XML for Analysis (XML/A) web service. All you need to do is deploy the .WAR file in whatever way is appropriate for your servlet container. After that, you can immediately explore Mondrian by playing with the sample applications and data, without any additional manual configuration.
